免密登录

  • URL中体现
  • SSH实现
  1. 生成公钥和私钥，默认放在~/.ssh目录下，id_rsa.pub公钥、id_rsa私钥。
    ssh-keygen
  2. 拷贝公钥的内容，并设置到GitHub中
  3. 在git本地中配置SSH地址
    git remote add origin git@github.com:WuPeiqi/dbhot.git
  4. 以后使用
    git push origin master
